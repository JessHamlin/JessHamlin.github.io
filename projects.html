Projects that I have worked on:

Snake Game
This project was quite fun. I created a version of the classic Snake Game that included an account system and numerous customizable options. This was done with python. The biggest part of this project was the NEAT ai system that allowed for the snake to be controlled by a computer, and allow enemy snakes to attack the player.
Number Guessing Game
In this project, the player must guess a number. The game will tell the player if they guessed above or below a desired number and let them guess again. If the player guess the desired number within a certain number of tries, they win. The big part of this project was custom random number generation.
Platformer Game
This project is available on my github account. It is a simple command line game in which a player traverses a two dimensional environment. It includes gravity and collision, and was my Data Structures class final project.
